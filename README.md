# SNEAKO - Ecommerce website for Sneaker Heads

This is the repository for my personal Sneakerspot website. The website is built using Astro 🚀, React, Typescript, TailwindCSS and is hosted on VERCEL - https://sneako.vercel.app

# Table of Contents

- [Installation](#installation)
- [Usage](#usage)

# Installation

To run the website locally, you'll need to clone the repository and type "npm install" in the terminal of the project folder. This will install all the necessary packages to run the program. Later you need to type "npm run dev" in the terminal and our site will be hosted locally on https://localhost:3000

# Usage

This is an E-commerce website for Sneaker lovers. Visitors can view my trending sneakers of companies like Nike, Puma and Adidas of different types. Also, there are running and football shoes availlable on the website.
